    问题描述

    你希望建立一个模型，能够根据房屋的大小来预测房价。

    让我们使用与上一个实验相同的两个数据点：一栋1000平方英尺的房屋售价为300,000美元，一栋2000平方英尺的房屋售价为500,000美元。

    面积 (1000平方英尺) 价格 (1000美元)
    1 300
    2 500 
    x_train = np.array([1.0, 2.0])           #(面积，单位：1000平方英尺)
y_train = np.array([300.0, 500.0])           #(价格，单位：1000美元)

    计算代价 (Computing Cost)

    本作业中的“代价（cost）”一词可能会有点令人困惑，因为数据本身就是房屋的成本（售价）。在这里，“代价”是衡量我们的模型预测目标房价准确程度的指标。房屋数据使用的是“价格（price）”一词。

    单变量代价函数的公式为：

    $$J(w,b) = \frac{1}{2m} \sum\limits_{i = 0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)})^2 \tag{1}$$

    其中

    $$f_{w,b}(x^{(i)}) = wx^{(i)} + b \tag{2}$$

    • $f_{w,b}(x^{(i)})$ 是使用参数 $w,b$ 对第 $i$ 个样本的预测值。

    • $(f_{w,b}(x^{(i)}) - y^{(i)})^2$ 是目标值与预测值之差的平方。

    • 将所有 $m$ 个样本的这些差值平方求和，然后除以 2m，即可得出代价 $J(w,b)$。

    注意：在课程讲座中，求和范围通常是从 1 到 m，而在代码中将是从 0 到 m-1。

    下面的代码通过循环遍历每个样本来计算代价。在每次循环中：

    • 计算预测值 f_wb

    • 计算目标值与预测值之间的差值并求平方。

    • 将其加到总代价中。

    def compute_cost(x, y, w, b): 
    """
    计算线性回归的代价函数。

    参数:
      x (ndarray (m,)): 数据，m个样本 
      y (ndarray (m,)): 目标值
      w,b (scalar)    : 模型参数  

    返回:
        total_cost (float): 使用 w,b 作为线性回归参数来拟合 x 和 y 中的数据点所产生的代价
    """
    # 训练样本的数量
    m = x.shape[0] 

    cost_sum = 0 
    for i in range(m): 
        f_wb = w * x[i] + b   
        cost = (f_wb - y[i]) ** 2  
        cost_sum = cost_sum + cost  
    total_cost = (1 / (2 * m)) * cost_sum  

    return total_cost

    代价函数直观理解

    ![[C1_W1_Lab02_GoalOfRegression.png]] 你的目标是找到一个具有参数 $w,b$ 的模型 $f_{w,b}(x) = wx + b$，该模型能够在给定输入 $x$ 的情况下准确预测房屋的价值。代价是衡量模型在训练数据上准确度的一个指标。

    上面的代价方程 (1) 表明，如果能选择合适的 $w$ 和 $b$，使得预测值 $f_{w,b}(x)$ 与目标数据 $y$ 匹配，那么 $(f_{w,b}(x^{(i)}) - y^{(i)})^2$ 项将为零，代价随之最小化。在这个简单的两点示例中，你可以完美实现这一点！

    在上一个实验中，你确定了 $b=100$ 能提供最佳解决方案，所以我们设 $b$ 为 100，并重点关注 $w$。

    plt_intuition(x_train,y_train)

    ![[Pasted image 20260511105042.png]] 该图包含几个值得注意的地方：

    • 当 $w = 200$ 时，代价最小化，这与上一个实验的结果一致。

    • 因为在代价方程中，目标值与预测值之间的差值被平方了，所以当 $w$ 过大或过小时，代价会迅速增加。

    • 使用通过最小化代价选出的 w 和 b 会得到一条完美拟合数据的直线。

    凸代价曲面

    代价函数对损失进行平方处理，这一事实确保了“误差曲面”像汤碗一样是凸的。它总会有一个最小值，通过沿着所有维度的梯度方向就能达到该最小值。在上一个图中，由于 $w$ 和 $b$ 的维度缩放比例不同，这一点不容易识别。在课程讲座中展示了下面这张 $w$ 和 $b$ 对称的图表：
