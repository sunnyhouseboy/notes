    分类

    在本实验中，你将对比回归和分类的区别。

    import numpy as np
%matplotlib widget
import matplotlib.pyplot as plt
from lab_utils_common import dlc, plot_data
from plt_one_addpt_onclick import plt_one_addpt_onclick
plt.style.use('./deeplearning.mplstyle')

    分类问题

    ![[C1_W3_Classification.png]] 分类问题的例子包括：将电子邮件识别为垃圾邮件或非垃圾邮件，或确定肿瘤是恶性还是良性。特别地，这些是二元分类的例子，即只有两种可能的结果。结果可以用“正”/“负”对来描述，例如“是”/“否”、“真”/“假”或“1”/“0”。

    分类数据集的图表通常使用符号来表示示例的结果。在下面的图表中，“X”用于表示正值，而“O”表示负结果。

    x_train = np.array([0., 1, 2, 3, 4, 5])
y_train = np.array([0,  0, 0, 1, 1, 1])
X_train2 = np.array([[0.5, 1.5], [1,1], [1.5, 0.5], [3, 0.5], [2, 2], [1, 2.5]])
y_train2 = np.array([0, 0, 0, 1, 1, 1])

    pos = y_train == 1#返回与 `y_train` 长度相同的**布尔型数组**（里面全是 `True` 和 `False`）
neg = y_train == 0

fig,ax = plt.subplots(1,2,figsize=(8,3))
#plot 1, single variable
ax[0].scatter(x_train[pos], y_train[pos], marker='x', s=80, c = 'red', label="y=1")
ax[0].scatter(x_train[neg], y_train[neg], marker='o', s=100, label="y=0", facecolors='none', 
              edgecolors=dlc["dlblue"],lw=3)

ax[0].set_ylim(-0.08,1.1)
ax[0].set_ylabel('y', fontsize=12)
ax[0].set_xlabel('x', fontsize=12)
ax[0].set_title('one variable plot')
ax[0].legend()

#plot 2, two variables
plot_data(X_train2, y_train2, ax[1])
ax[1].axis([0, 4, 0, 4])
ax[1].set_ylabel('$x_1$', fontsize=12)
ax[1].set_xlabel('$x_0$', fontsize=12)
ax[1].set_title('two variable plot')
ax[1].legend()
plt.tight_layout()
plt.show()

    ![[Pasted image 20260512110645.png]] 请注意在上面的图表中：

    • 在单变量图表中，正结果既显示为红色“X”，也位于 y=1 处。负结果是蓝色的“O”，位于 y=0 处。

      • 回想一下，在线性回归的情况下，y 不会被限制在两个值，而是可以是任何值。

    • 在双变量图表中，y 轴不可用。正结果显示为红色“X”，而负结果使用蓝色“O”符号。

      • 回想一下，在具有多个变量的线性回归情况下，y 不会被限制为两个值，并且类似的图表将是三维的。

    线性回归方法

    在上一周，你应用了线性回归来构建预测模型。让我们在这里使用课程中描述的简单示例来尝试该方法。该模型将根据肿瘤大小预测肿瘤是良性还是恶性。请尝试以下操作：

    • 点击“运行线性回归（Run Linear Regression）”以为给定数据找到最佳的线性回归模型。

      • 请注意，生成的线性模型与数据的匹配度不好。

    • 一种改善结果的方法是应用一个阈值。

    • 勾选“切换 0.5 阈值（Toggle 0.5 threshold）”复选框，以显示应用阈值后的预测结果。

      • 这些预测看起来不错，预测与数据相匹配。

    • 重要提示：现在，在最右侧大肿瘤大小范围（接近 10）内添加更多“恶性”数据点，并重新运行线性回归。

      • 现在，模型预测了较大的肿瘤，但 x=3 处的数据点被错误预测了！

    • 要清除/更新绘图，请重新运行包含绘图命令的单元格。
