### 计算代价 (Compute_Cost)

这个函数在上一个实验中已经开发过了。我们在这里将再次需要它。


```python
# 计算代价的函数
def compute_cost(x, y, w, b):
   
    m = x.shape[0] 
    cost = 0
    
    for i in range(m):
        f_wb = w * x[i] + b
        cost = cost + (f_wb - y[i])**2
    total_cost = 1 / (2 * m) * cost

    return total_cost
```

## 梯度下降总结 (Gradient descent summary)

到目前为止，在本课程中，你已经开发了一个预测 $f_{w,b}(x^{(i)})$ 的线性模型：

$$f_{w,b}(x^{(i)}) = wx^{(i)} + b \tag{1}$$

在线性回归中，你利用输入的训练数据来拟合参数 $w$,$b$，方法是最小化我们的预测值 $f_{w,b}(x^{(i)})$ 和实际数据 $y^{(i)}$ 之间误差的度量。这个度量被称为代价 (cost)，即 $J(w,b)$。在训练中，你测量所有训练样本 $x^{(i)},y^{(i)}$ 上的总代价：

$$J(w,b) = \frac{1}{2m} \sum\limits_{i = 0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)})^2\tag{2}$$

在讲座中，_梯度下降 (gradient descent)_ 被描述为：

$$\begin{align*} \text{repeat}&\text{ until convergence:} ; \lbrace \newline

; w &= w - \alpha \frac{\partial J(w,b)}{\partial w} \tag{3} ; \newline

b &= b - \alpha \frac{\partial J(w,b)}{\partial b} \newline \rbrace

\end{align*}$$

其中，参数 $w$, $b$ 被同时更新。

梯度定义为：

$$\begin{align} \frac{\partial J(w,b)}{\partial w} &= \frac{1}{m} \sum\limits_{i = 0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)})x^{(i)} \tag{4}\\ \frac{\partial J(w,b)}{\partial b} &= \frac{1}{m} \sum\limits_{i = 0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)}) \tag{5}\\ \end{align}$$

这里的_同时 (simultaneously)_ 意味着在更新任何一个参数之前，你要先计算出所有参数的偏导数。

## 实现梯度下降 (Implement Gradient Descent)

你将实现单特征的梯度下降算法。你将需要三个函数：

- `compute_gradient`：实现上方的方程 (4) 和 (5)
    
- `compute_cost`：实现上方的方程 (2) (代码来自上一个实验)
    
- `gradient_descent`：调用 compute_gradient 和 compute_cost
    

约定：

- 包含偏导数的 python 变量命名遵循这个模式，例如 $\frac{\partial J(w,b)}{\partial b}$ 将会被命名为 `dj_db`。
    
- w.r.t 表示“关于 (With Respect To)”，比如 $J(wb)$ 关于 $b$ 的偏导数。
    

### 计算梯度 (compute_gradient)

`compute_gradient` 实现上方的 (4) 和 (5) 并返回 $\frac{\partial J(w,b)}{\partial w}$,$\frac{\partial J(w,b)}{\partial b}$。内嵌注释描述了具体操作过程。

```python
def compute_gradient(x, y, w, b): 
    """
    计算线性回归的梯度 
    参数 (Args):
      x (ndarray (m,)): 数据，m个样本 
      y (ndarray (m,)): 目标值
      w,b (scalar)    : 模型参数  
    返回 (Returns):
      dj_dw (scalar): 代价关于参数 w 的梯度
      dj_db (scalar): 代价关于参数 b 的梯度     
     """
    
    # 训练样本的数量
    m = x.shape[0]    
    dj_dw = 0
    dj_db = 0
    
    for i in range(m):  
        f_wb = w * x[i] + b 
        dj_dw_i = (f_wb - y[i]) * x[i] 
        dj_db_i = f_wb - y[i] 
        dj_db += dj_db_i
        dj_dw += dj_dw_i 
    dj_dw = dj_dw / m 
    dj_db = dj_db / m 
        
    return dj_dw, dj_db
```

请注意，梯度指向_远离_最小值的方向。回顾上面的方程 (3)，缩放后的梯度是从 $w$ 或 $b$ 的当前值中_减去_的。这会使参数朝着降低代价的方向移动。

### 梯度下降 (Gradient Descent)

既然可以计算梯度了，上面方程 (3) 描述的梯度下降现在可以在下面的 `gradient_descent` 函数中实现了。实现的细节在注释中有描述。在下方，你将利用此函数来寻找训练数据中 $w$ 和 $b$ 的最优值。
```python
def gradient_descent(x, y, w_in, b_in, alpha, num_iters, cost_function, gradient_function): 
    """
    执行梯度下降来拟合 w,b。通过采用学习率 alpha 采取 num_iters 个梯度步长来更新 w,b。
    
    参数 (Args):
      x (ndarray (m,))  : 数据，m个样本 
      y (ndarray (m,))  : 目标值
      w_in,b_in (scalar): 模型参数的初始值  
      alpha (float):     学习率
      num_iters (int):   运行梯度下降的迭代次数
      cost_function:     计算代价的函数调用
      gradient_function: 计算梯度的函数调用
      
    返回 (Returns):
      w (scalar): 运行梯度下降后更新的参数值
      b (scalar): 运行梯度下降后更新的参数值
      J_history (List): 代价值的历史记录
      p_history (list): 参数 [w,b] 的历史记录 
      """
    
    w = copy.deepcopy(w_in) # 避免修改全局变量 w_in
    # 存储每次迭代的代价 J 和 w 的数组，主要用于之后绘图
    J_history = []
    p_history = []
    b = b_in
    w = w_in
    
    for i in range(num_iters):
        # 计算梯度并使用 gradient_function 更新参数
        dj_dw, dj_db = gradient_function(x, y, w , b)     

        # 使用上方方程 (3) 更新参数
        b = b - alpha * dj_db                            
        w = w - alpha * dj_dw                            

        # 保存每次迭代的代价 J
        if i<100000:      # 防止资源耗尽 
            J_history.append( cost_function(x, y, w , b))
            p_history.append([w,b])
        # 每隔 10% 的迭代次数打印一次代价（如果迭代次数 < 10，则每次都打印）
        if i% math.ceil(num_iters/10) == 0:
            print(f"Iteration {i:4}: Cost {J_history[-1]:0.2e} ",
                  f"dj_dw: {dj_dw: 0.3e}, dj_db: {dj_db: 0.3e}  ",
                  f"w: {w: 0.3e}, b:{b: 0.5e}")
 
    return w, b, J_history, p_history #返回 w, J 和 p 的历史记录用于绘图
```
