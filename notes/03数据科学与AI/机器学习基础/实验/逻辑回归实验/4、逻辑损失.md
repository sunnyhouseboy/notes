## 逻辑回归使用平方误差？ 
![[C1_W3_SqErrorVsLogistic.png]]
 回顾一下，在**线性**回归中我们使用了**平方误差代价函数**：
单变量平方误差代价方程为：
  $$J(w,b) = \frac{1}{2m} \sum\limits_{i = 0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)})^2 \tag{1}$$ 
 
其中 
  $$f_{w,b}(x^{(i)}) = wx^{(i)} + b \tag{2}$$


回顾一下，平方误差代价具有一个很好的特性，即沿着代价的导数方向能够找到最小值。


```python
soup_bowl()
```

这个代价函数在线性回归中表现很好，很自然地我们会考虑将其也用于逻辑回归。然而，正如上面的幻灯片所指出的，现在 $f_{wb}(x)$ 有了一个非线性组件，即 sigmoid 函数：$f_{w,b}(x^{(i)}) = sigmoid(wx^{(i)} + b )$。让我们在之前实验的例子上尝试使用平方误差代价，现在包含了 sigmoid 函数。

这里是我们的训练数据：


```python
x_train = np.array([0., 1, 2, 3, 4, 5],dtype=np.longdouble)
y_train = np.array([0,  0, 0, 1, 1, 1],dtype=np.longdouble)
plt_simple_example(x_train, y_train)
```

现在，让我们使用*平方误差代价*来绘制代价的表面图（surface plot）：
  $$J(w,b) = \frac{1}{2m} \sum\limits_{i = 0}^{m-1} (f_{w,b}(x^{(i)}) - y^{(i)})^2 $$ 
 
其中 
  $$f_{w,b}(x^{(i)}) = sigmoid(wx^{(i)} + b )$$


```python

```

逻辑回归需要一个更适合其非线性特性的代价函数。这从损失函数（Loss function）开始。下面将对此进行描述。

## 逻辑损失函数 
![[C1_W3_LogisticLoss_a.png]]
![[C1_W3_LogisticLoss_b.png]]
![[C1_W3_LogisticLoss_c.png]]

逻辑回归使用了一种更适合分类任务的损失函数，其目标值是0或1而不是任意数字。

>**定义提示：** 在本课程中，使用了以下定义：
>**损失 (Loss)** 是单一示例与其目标值之间差异的度量，而
>**代价 (Cost)** 是整个训练集上损失的度量。

其定义如下：
* $loss(f_{\mathbf{w},b}(\mathbf{x}^{(i)}), y^{(i)})$ 是单一数据点的代价，即：
$$\begin{equation}
\text{loss}(f_{\mathbf{w},b}(\mathbf{x}^{(i)}), y^{(i)}) = \begin{cases}
  - \log\left(f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right) & \text{if } y^{(i)} = 1\\
  - \log \left( 1 - f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right) & \text{if } y^{(i)} = 0
\end{cases}
\end{equation}$$
* $f_{\mathbf{w},b}(\mathbf{x}^{(i)})$ 是模型的预测值，而 $y^{(i)}$ 是目标值。
* $f_{\mathbf{w},b}(\mathbf{x}^{(i)}) = g(\mathbf{w} \cdot\mathbf{x}^{(i)}+b)$，其中函数 $g$ 是 sigmoid 函数。

这个损失函数的决定性特征是它使用了两条不同的曲线。一条用于目标为0（或 $y=0$）的情况，另一条用于目标为1（$y=1$）的情况。结合起来，这些曲线提供了一个对损失函数有用的行为，即当预测匹配目标时为零，而当预测偏离目标时值快速增加。考虑下面的曲线：


```python
plt_two_logistic_loss_curves()
```
![[Pasted image 20260512154826.png]]
结合起来，这些曲线类似于平方误差损失的二次曲线。注意，x轴是 $f_{\mathbf{w},b}$，即 sigmoid 的输出。sigmoid 的输出严格介于 0 和 1 之间。


上面的损失函数可以重写以便更容易实现：
    $$loss(f_{\mathbf{w},b}(\mathbf{x}^{(i)}), y^{(i)}) = (-y^{(i)} \log\left(f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right) - \left( 1 - y^{(i)}\right) \log \left( 1 - f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right)$$
  
这是一个看起来相当复杂的方程。但是考虑到 $y^{(i)}$ 只能有两个值，0 和 1 时，就不那么令人生畏了。然后我们可以将方程分成两部分来考虑：
当 $y^{(i)} = 0$ 时，左边的项被消除：
$$
\begin{align}
loss(f_{\mathbf{w},b}(\mathbf{x}^{(i)}), 0) &= (-(0) \log\left(f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right) - \left( 1 - 0\right) \log \left( 1 - f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right) \\
&= -\log \left( 1 - f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right)
\end{align}
$$
当 $y^{(i)} = 1$ 时，右边的项被消除：
$$
\begin{align}
  loss(f_{\mathbf{w},b}(\mathbf{x}^{(i)}), 1) &=  (-(1) \log\left(f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right) - \left( 1 - 1\right) \log \left( 1 - f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right)\\
  &=  -\log\left(f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right)
\end{align}
$$

好了，有了这个新的逻辑损失函数，我们就可以产生一个合并了所有示例损失的代价函数。这将是下一个实验的主题。现在，让我们看看我们在上面考虑的简单示例的代价与参数曲线：


```python
plt.close('all')
cst = plt_logistic_cost(x_train,y_train)
```
