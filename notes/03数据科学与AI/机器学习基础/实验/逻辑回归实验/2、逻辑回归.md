## Sigmoid 函数或逻辑函数
![[C1_W3_LogisticRegression_left.png]]
正如在教学视频中所讨论的，对于分类任务，我们可以首先使用我们的线性回归模型 $f_{\mathbf{w},b}(\mathbf{x}^{(i)}) = \mathbf{w} \cdot \mathbf{x}^{(i)} + b$，在给定 $x$ 的情况下预测 $y$。

- 然而，我们希望分类模型的预测值在 0 和 1 之间，因为我们的输出变量 $y$ 只能是 0 或 1。
    
- 这可以通过使用“sigmoid 函数”来实现，该函数将所有输入值映射到 0 和 1 之间的值。
    

让我们自己实现 sigmoid 函数并观察一下。

## Sigmoid 函数公式

sigmoid 函数的公式如下：

$$g(z) = \frac{1}{1+e^{-z}}$$

在逻辑回归的情况下，$z$（sigmoid 函数的输入）是线性回归模型的输出。

- 在单个样本的情况下，$z$ 是一个标量。
    
- 在多个样本的情况下，$z$ 可能是一个包含 $m$ 个值的向量，每个值对应一个样本。
    
- sigmoid 函数的实现应该涵盖这两种潜在的输入格式。
    

让我们在 Python 中实现它。

NumPy 有一个名为 `exp()` 的函数，它提供了一种方便的方法来计算输入数组（`z`）中所有元素的指数（$e^{z}$）。

它也可以接受单个数字作为输入，如下所示。

```python
# 输入是一个数组。 
input_array = np.array([1,2,3])
exp_array = np.exp(input_array)

print("Input to exp:", input_array)
print("Output of exp:", exp_array)

# 输入是一个单一数字
input_val = 1  
exp_val = np.exp(input_val)

print("Input to exp:", input_val)
print("Output of exp:", exp_val)
```

`sigmoid` 函数在 Python 中的实现如下方单元格所示。

```python
def sigmoid(z):
    """
    计算 z 的 sigmoid 值

    参数:
        z (ndarray): 任意大小的标量或 numpy 数组。

    返回:
        g (ndarray): sigmoid(z)，具有与 z 相同的形状
         
    """

    g = 1/(1+np.exp(-z))
   
    return g
```

让我们看看对于不同的 `z` 值，这个函数的输出是什么：

```python
# 生成一个包含 -10 到 10 之间均匀分布值的数组
z_tmp = np.arange(-10,11)

# 使用上面实现的函数获取 sigmoid 值
y = sigmoid(z_tmp)

# 用于美观地将两个数组并排打印的代码
np.set_printoptions(precision=3) 
print("Input (z), Output (sigmoid(z))")
print(np.c_[z_tmp, y])
```

左列的值是 `z`，右列的值是 `sigmoid(z)`。正如你所看到的，sigmoid 的输入值范围从 -10 到 10，而输出值范围从 0 到 1。

现在，让我们尝试使用 `matplotlib` 库来绘制这个函数。

```python
# 绘制 z 与 sigmoid(z) 的关系图
fig,ax = plt.subplots(1,1,figsize=(5,3))
ax.plot(z_tmp, y, c="b")

ax.set_title("Sigmoid function")
ax.set_ylabel('sigmoid(z)')
ax.set_xlabel('z')
draw_vthresh(ax,0)
```

正如你所看到的，当 `z` 趋向于很大的负值时，sigmoid 函数趋近于 **0**；当 `z` 趋向于很大的正值时，sigmoid 函数趋近于 **1**。

## 逻辑回归
![[C1_W3_LogisticRegression_right.png]]
逻辑回归模型将 sigmoid 函数应用于我们熟悉的线性回归模型，如下所示：

$$f_{\mathbf{w},b}(\mathbf{x}^{(i)}) = g(\mathbf{w} \cdot \mathbf{x}^{(i)} + b )$$

其中

$$g(z) = \frac{1}{1+e^{-z}}$$

让我们将逻辑回归应用于肿瘤分类的分类数据示例。
