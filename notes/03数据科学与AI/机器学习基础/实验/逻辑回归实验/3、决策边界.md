## 逻辑回归模型

- 假设你想根据这些数据训练一个逻辑回归模型，其形式为
    
    $$f(x) = g(w_0x_0+w_1x_1 + b)$$
    
    其中 $g(z) = \frac{1}{1+e^{-z}}$，即 sigmoid 函数
    
- 假设你已经训练了模型，得到的参数为 $b = -3$，$w_0 = 1$，$w_1 = 1$。 也就是说，
    
    $$f(x) = g(x_0+x_1-3)$$
    
    （在后续课程中，你将学习如何根据数据拟合这些参数）
    

让我们尝试通过绘制决策边界来理解这个训练好的模型正在预测什么。

### 逻辑回归与决策边界复习

- 回顾一下，对于逻辑回归，模型表示为
    
    $$f_{\mathbf{w},b}(\mathbf{x}^{(i)}) = g(\mathbf{w} \cdot \mathbf{x}^{(i)} + b) \tag{1}$$
    
    其中 $g(z)$ 被称为 sigmoid 函数，它将所有输入值映射到 0 和 1 之间的值：
    
    $$g(z) = \frac{1}{1+e^{-z}}\tag{2}$$
    
    并且 $\mathbf{w} \cdot \mathbf{x}$ 是向量点积：
    
    $$\mathbf{w} \cdot \mathbf{x} = w_0 x_0 + w_1 x_1$$
    
- 我们将模型的输出（$f_{\mathbf{w},b}(x)$）解释为在给定 $\mathbf{x}$ 且由 $\mathbf{w}$ 和 $b$ 参数化的情况下，预测 $y=1$ 的概率。
    
- 因此，为了从逻辑回归模型获得最终预测（$y=0$ 或 $y=1$），我们可以使用以下启发式方法 -
    
    如果 $f_{\mathbf{w},b}(x) \ge 0.5$，预测 $y=1$
    
    如果 $f_{\mathbf{w},b}(x) < 0.5$，预测 $y=0$
    
- 让我们绘制 sigmoid 函数以查看 $g(z) \ge 0.5$ 的位置
    
```python
# 在 -10 到 10 的值范围内绘制 sigmoid(z)
z = np.arange(-10,11)

fig,ax = plt.subplots(1,1,figsize=(5,3))
# 绘制 z 与 sigmoid(z) 的关系图
ax.plot(z, sigmoid(z), c="b")

ax.set_title("Sigmoid function")
ax.set_ylabel('sigmoid(z)')
ax.set_xlabel('z')
draw_vthresh(ax,0)
```
![[notes/03数据科学与AI/机器学习基础/实验/逻辑回归实验/img/Figure 2.png]]
- 如你所见，当 $z \ge 0$ 时，$g(z) \ge 0.5$
    
- 对于逻辑回归模型，$z = \mathbf{w} \cdot \mathbf{x} + b$。 因此，
    
    如果 $\mathbf{w} \cdot \mathbf{x} + b \ge 0$，模型预测 $y=1$
    
    如果 $\mathbf{w} \cdot \mathbf{x} + b < 0$，模型预测 $y=0$
    

### 绘制决策边界

现在，让我们回到我们的示例，以了解逻辑回归模型是如何进行预测的。

- 我们的逻辑回归模型形式为
    
    $$f(\mathbf{x}) = g(-3 + x_0+x_1)$$
    
- 从上面的学习中，你可以看出如果 $-3 + x_0+x_1 \ge 0$，该模型预测 $y=1$
    

让我们看看它在图形上是什么样子的。 我们将从绘制 $-3 + x_0+x_1 = 0$ 开始，这等同于 $x_1 = 3 - x_0$。

```python
# 选择 0 到 6 之间的值
x0 = np.arange(0,6)

x1 = 3 - x0
fig,ax = plt.subplots(1,1,figsize=(5,4))
# 绘制决策边界
ax.plot(x0,x1, c="b")
ax.axis([0, 4, 0, 3.5])

# 填充直线下方区域
ax.fill_between(x0,x1, alpha=0.2)

# 绘制原始数据
plot_data(X,y,ax)
ax.set_ylabel(r'$x_1$')
ax.set_xlabel(r'$x_0$')
plt.show()
```
![[Pasted image 20260512151131.png]]
- 在上图中，蓝线代表直线 $x_0 + x_1 - 3 = 0$，它应该与 $x_1$ 轴在 3 处相交（如果我们设置 $x_1 = 3$，$x_0 = 0$），并与 $x_0$ 轴在 3 处相交（如果我们设置 $x_1 = 0$，$x_0 = 3$）。
    
- 阴影区域表示 $-3 + x_0+x_1 < 0$。 直线上方的区域是 $-3 + x_0+x_1 > 0$。
    
- 阴影区域（直线下方）中的任何点都被分类为 $y=0$。 直线上的或上方的任何点都被分类为 $y=1$。 这条线被称为“决策边界”。
    

正如我们在课程中看到的，通过使用高阶多项式项（例如：$f(x) = g( x_0^2 + x_1 -1)$），我们可以得出更复杂的非线性边界。
