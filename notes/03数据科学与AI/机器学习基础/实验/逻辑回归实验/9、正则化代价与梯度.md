## 包含正则化的代价函数

### 正则化线性回归的代价函数

正则化线性回归代价函数的方程为：

$$J(\mathbf{w},b) = \frac{1}{2m} \sum\limits_{i = 0}^{m-1} (f_{\mathbf{w},b}(\mathbf{x}^{(i)}) - y^{(i)})^2 + \frac{\lambda}{2m} \sum_{j=0}^{n-1} w_j^2 \tag{1}$$

其中：

$$f_{\mathbf{w},b}(\mathbf{x}^{(i)}) = \mathbf{w} \cdot \mathbf{x}^{(i)} + b \tag{2}$$

将其与没有正则化的代价函数（你在之前的实验中实现过）进行比较，其形式为：

$$J(\mathbf{w},b) = \frac{1}{2m} \sum\limits_{i = 0}^{m-1} (f_{\mathbf{w},b}(\mathbf{x}^{(i)}) - y^{(i)})^2 $$

区别在于正则化项，$\frac{\lambda}{2m} \sum_{j=0}^{n-1} w_j^2$

包含此项会鼓励梯度下降去最小化参数的数值大小。请注意，在这个例子中，参数 $b$ 没有被正则化。这是行业内的标准做法。

下面是方程 (1) 和 (2) 的实现代码。请注意，这里使用了 _本课程的标准模式_，即遍历所有 `m` 个样本的 `for` 循环。

```python
def compute_cost_linear_reg(X, y, w, b, lambda_ = 1):
    """
    计算所有样本的代价
    参数:
      X (ndarray (m,n)): 数据，m 个样本，n 个特征
      y (ndarray (m,)): 目标值
      w (ndarray (n,)): 模型参数  
      b (标量)        : 模型参数
      lambda_ (标量): 控制正则化的程度
    返回:
      total_cost (标量): 总代价 
    """

    m  = X.shape[0]
    n  = len(w)
    cost = 0.
    for i in range(m):
        f_wb_i = np.dot(X[i], w) + b                                  #(n,)(n,)=标量, 参见 np.dot
        cost = cost + (f_wb_i - y[i])**2                               #标量             
    cost = cost / (2 * m)                                              #标量  
 
    reg_cost = 0
    for j in range(n):
        reg_cost += (w[j]**2)                                          #标量
    reg_cost = (lambda_/(2*m)) * reg_cost                              #标量
    
    total_cost = cost + reg_cost                                       #标量
    return total_cost                                                  #标量
```

### 正则化逻辑回归的代价函数

对于正则化**逻辑**回归，代价函数的形式为

$$J(\mathbf{w},b) = \frac{1}{m} \sum_{i=0}^{m-1} \left[ -y^{(i)} \log\left(f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right) - \left( 1 - y^{(i)}\right) \log \left( 1 - f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right) \right] + \frac{\lambda}{2m} \sum_{j=0}^{n-1} w_j^2 \tag{3}$$

其中：

$$f_{\mathbf{w},b}(\mathbf{x}^{(i)}) = sigmoid(\mathbf{w} \cdot \mathbf{x}^{(i)} + b) \tag{4}$$

将其与没有正则化的代价函数（你在之前的实验中实现过）进行比较：

$$J(\mathbf{w},b) = \frac{1}{m}\sum_{i=0}^{m-1} \left[ (-y^{(i)} \log\left(f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right) - \left( 1 - y^{(i)}\right) \log \left( 1 - f_{\mathbf{w},b}\left( \mathbf{x}^{(i)} \right) \right)\right]$$

与上面的线性回归情况相同，区别在于正则化项，即 $\frac{\lambda}{2m} \sum_{j=0}^{n-1} w_j^2$

包含此项会鼓励梯度下降去最小化参数的大小。同样在这个例子中，参数 $b$ 没有被正则化。

```python
def compute_cost_logistic_reg(X, y, w, b, lambda_ = 1):
    """
    计算所有样本的代价
    参数:
      X (ndarray (m,n)): 数据，m 个样本，n 个特征
      y (ndarray (m,)): 目标值
      w (ndarray (n,)): 模型参数  
      b (标量)        : 模型参数
      lambda_ (标量): 控制正则化的程度
    返回:
      total_cost (标量): 总代价 
    """

    m,n  = X.shape
    cost = 0.
    for i in range(m):
        z_i = np.dot(X[i], w) + b                                      #(n,)(n,)=标量, 参见 np.dot
        f_wb_i = sigmoid(z_i)                                          #标量
        cost +=  -y[i]*np.log(f_wb_i) - (1-y[i])*np.log(1-f_wb_i)      #标量
             
    cost = cost/m                                                      #标量

    reg_cost = 0
    for j in range(n):
        reg_cost += (w[j]**2)                                          #标量
    reg_cost = (lambda_/(2*m)) * reg_cost                              #标量
    
    total_cost = cost + reg_cost                                       #标量
    return total_cost                                                  #标量
```

## 包含正则化的梯度下降

运行梯度下降的基本算法在加入正则化后并没有改变，它仍然是：

$$\begin{align*}

&\text{重复直到收敛:} ; \lbrace \

& ; ; ;w_j = w_j - \alpha \frac{\partial J(\mathbf{w},b)}{\partial w_j} \tag{1} ; & \text{for j := 0..n-1} \

& ; ; ; ; ;b = b - \alpha \frac{\partial J(\mathbf{w},b)}{\partial b} \

&\rbrace

\end{align*}$$

在每次迭代中，所有的 $w_j$ 都会被同时更新。

加入正则化后改变的是梯度的计算方式。

### 计算带正则化的梯度 (线性/逻辑回归)

线性回归和逻辑回归的梯度计算几乎完全相同，唯一的区别在于 $f_{\mathbf{w}b}$ 的计算方式。

$$\begin{align*}

\frac{\partial J(\mathbf{w},b)}{\partial w_j} &= \frac{1}{m} \sum\limits_{i = 0}^{m-1} (f_{\mathbf{w},b}(\mathbf{x}^{(i)}) - y^{(i)})x_{j}^{(i)} + \frac{\lambda}{m} w_j \tag{2} \\

\frac{\partial J(\mathbf{w},b)}{\partial b} &= \frac{1}{m} \sum\limits_{i = 0}^{m-1} (f_{\mathbf{w},b}(\mathbf{x}^{(i)}) - y^{(i)}) \tag{3}

\end{align*}$$

- `m` 是数据集中的训练样本数量
    
- $f_{\mathbf{w},b}(x^{(i)})$ 是模型的预测值，而 $y^{(i)}$ 是真实的目标值
    
- 对于 **线性** 回归模型
    
    $f_{\mathbf{w},b}(x) = \mathbf{w} \cdot \mathbf{x} + b$
    
- 对于 **逻辑** 回归模型
    
    $z = \mathbf{w} \cdot \mathbf{x} + b$
    
    $f_{\mathbf{w},b}(x) = g(z)$
    
    其中 $g(z)$ 是 sigmoid 函数:
    
    $g(z) = \frac{1}{1+e^{-z}}$
    

添加正则化的项是 $\frac{\lambda}{m} w_j$。

### 正则化线性回归的梯度函数

```python
def compute_gradient_linear_reg(X, y, w, b, lambda_): 
    """
    计算线性回归的梯度 
    参数:
      X (ndarray (m,n)): 数据，m 个样本，n 个特征
      y (ndarray (m,)): 目标值
      w (ndarray (n,)): 模型参数  
      b (标量)        : 模型参数
      lambda_ (标量): 控制正则化的程度
      
    返回:
      dj_dw (ndarray (n,)): 代价关于参数 w 的梯度。 
      dj_db (标量):         代价关于参数 b 的梯度。 
    """
    m,n = X.shape           #(样本数量, 特征数量)
    dj_dw = np.zeros((n,))
    dj_db = 0.

    for i in range(m):                             
        err = (np.dot(X[i], w) + b) - y[i]                 
        for j in range(n):                         
            dj_dw[j] = dj_dw[j] + err * X[i, j]               
        dj_db = dj_db + err                        
    dj_dw = dj_dw / m                                
    dj_db = dj_db / m   
    
    for j in range(n):
        dj_dw[j] = dj_dw[j] + (lambda_/m) * w[j]

    return dj_db, dj_dw
```

### 正则化逻辑回归的梯度函数

```python
def compute_gradient_logistic_reg(X, y, w, b, lambda_): 
    """
    计算逻辑回归的梯度 
 
    参数:
      X (ndarray (m,n)): 数据，m 个样本，n 个特征
      y (ndarray (m,)): 目标值
      w (ndarray (n,)): 模型参数  
      b (标量)        : 模型参数
      lambda_ (标量): 控制正则化的程度
    返回:
      dj_dw (ndarray 形状 (n,)): 代价关于参数 w 的梯度。 
      dj_db (标量)             : 代价关于参数 b 的梯度。 
    """
    m,n = X.shape
    dj_dw = np.zeros((n,))                            #(n,)
    dj_db = 0.0                                       #标量

    for i in range(m):
        f_wb_i = sigmoid(np.dot(X[i],w) + b)          #(n,)(n,)=标量
        err_i  = f_wb_i  - y[i]                       #标量
        for j in range(n):
            dj_dw[j] = dj_dw[j] + err_i * X[i,j]      #标量
        dj_db = dj_db + err_i
    dj_dw = dj_dw/m                                   #(n,)
    dj_db = dj_db/m                                   #标量

    for j in range(n):
        dj_dw[j] = dj_dw[j] + (lambda_/m) * w[j]

    return dj_db, dj_dw  
```
