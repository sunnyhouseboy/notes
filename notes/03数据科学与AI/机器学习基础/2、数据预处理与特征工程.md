# 📚 《菜菜的机器学习sklearn课堂》笔记：数据预处理与特征工程

## 🌟 0. 概述

在机器学习中，数据和特征决定了机器学习的上限，而模型和算法只是逼近这个上限而已。

- **数据预处理 (Data Preprocessing)**：属于数据准备阶段。目的是让数据适应模型，匹配模型的需求（如：无量纲化、填补缺失值、编码分类型变量等）。
    
- **特征工程 (Feature Engineering)**：将原始数据转化为更能代表预测模型潜在问题的特征的过程。主要包括：**特征提取 (Feature Extraction)**、**特征创造 (Feature Creation)** 和 **特征选择 (Feature Selection)**。

## 🛠️ 1. 数据预处理 (Data Preprocessing)

`sklearn` 中主要通过 `sklearn.preprocessing` 模块进行数据预处理。

### 1.1 数据无量纲化

将不同规格的数据转换到同一规格，或不同分布的数据转换到某个特定分布。主要目的是加快求解速度、提升模型精度（尤其是对于距离类模型如 KNN、KMeans、SVM，以及梯度下降法求解的模型）。

#### 1.1.1 归一化 (Normalization / Min-Max Scaling)

- **原理**：将数据压缩到 `[0, 1]` 之间（或者其他指定范围）。受异常值影响较大。
    
- **模块**：`sklearn.preprocessing.MinMaxScaler`
    
- **核心代码**：
    
    Python
    
    ```
    from sklearn.preprocessing import MinMaxScaler
    import pandas as pd
    
    data = [[-1, 2], [-0.5, 6], [0, 10], [1, 18]]
    # 1. 实例化
    scaler = MinMaxScaler()  # 也可指定 feature_range=[0,1]
    # 2. fit: 本质是生成min(x)和max(x)
    scaler = scaler.fit(data) 
    # 3. transform: 导出结果
    result = scaler.transform(data) 
    
    # fit_transform 一步达成
    result_ = scaler.fit_transform(data)
    
    # 逆转标准化
    scaler.inverse_transform(result)
    ```
    

#### 1.1.2 标准化 (Standardization)

- **原理**：将数据按均值(mean)中心化后，再按标准差(std)缩放，数据就会服从均值为0，方差为1的正态分布。
    
- **模块**：`sklearn.preprocessing.StandardScaler`
    
- **核心代码**：
    
    Python
    
    ```
    from sklearn.preprocessing import StandardScaler
    
    scaler = StandardScaler()
    scaler.fit(data) # fit本质是求出均值和方差
    print(scaler.mean_) # 查看均值
    print(scaler.var_)  # 查看方差
    
    x_std = scaler.transform(data) # 导出结果
    ```
    

> **📌 总结：归一化 vs 标准化**
> 
> 大多数机器学习算法中，会选择**StandardScaler**来进行特征缩放，因为MinMaxScaler对异常值非常敏感。在PCA、聚类、逻辑回归、SVM、神经网络中，StandardScaler表现往往更好。MinMaxScaler主要用于不涉及距离度量、梯度、并且对数据分布有严格范围要求的场景（如图像处理）。

### 1.2 缺失值处理

机器学习模型通常无法处理包含缺失值（NaN）的矩阵，因此需要填补或删除。

#### 1.2.1 使用 sklearn 处理 (`SimpleImputer`)

- **模块**：`sklearn.impute.SimpleImputer`
    
- **参数**：
    
    - `missing_values`：告诉imputer什么是缺失值，默认 `np.nan`
        
    - `strategy`：填补策略，可选 `"mean"`(均值), `"median"`(中位数), `"most_frequent"`(众数), `"constant"`(常数)
        
    - `fill_value`：当 strategy="constant" 时，填补的具体数值。
        
- **核心代码**：
    
    Python
    
    ```
    from sklearn.impute import SimpleImputer
    
    # 假设 Age 列有缺失值
    Age = data.loc[:,"Age"].values.reshape(-1,1) # sklearn要求传入二维数组
    
    imp_mean = SimpleImputer() # 默认均值填补
    imp_median = SimpleImputer(strategy="median") # 中位数填补
    imp_0 = SimpleImputer(strategy="constant", fill_value=0) # 用0填补
    
    imp_mean = imp_mean.fit_transform(Age)
    ```
    

#### 1.2.2 使用 Pandas 处理（更推荐，快捷）

Python

```
# 填补缺失值
data.loc[:,"Age"] = data.loc[:,"Age"].fillna(data.loc[:,"Age"].median())
# 丢弃包含缺失值的行
data.dropna(axis=0, inplace=True)
```

### 1.3 处理分类型特征：编码与哑变量

机器学习大多数算法只能处理数字，不能处理文字。因此我们需要把分类数据转为数值。

#### 1.3.1 标签编码 (LabelEncoder) - 专用于标签 `y`

- **模块**：`sklearn.preprocessing.LabelEncoder`
    
- **核心代码**：
    
    Python
    
   ```python
    from sklearn.preprocessing import LabelEncoder
    
    y = data.iloc[:,-1] # 假设最后一列是标签
    le = LabelEncoder()
    le = le.fit(y)
    label = le.transform(y)
    
    # 快捷写法
    data.iloc[:,-1] = LabelEncoder().fit_transform(data.iloc[:,-1])
    ```
    

#### 1.3.2 特征编码 (OrdinalEncoder) - 专用于特征 `X` (类别之间有大小/等级关系)

- **模块**：`sklearn.preprocessing.OrdinalEncoder`
    
- 比如：学历（小学、初中、高中、大学）有高低之分，适合转换为 0, 1, 2, 3。
    

#### 1.3.3 独热编码 / 哑变量 (OneHotEncoder) - 专用于名义变量

- **原理**：大多数分类特征（如：性别[男, 女]、颜色[红, 黄, 蓝]）是名义变量，相互之间没有大小关系。如果直接用 0,1,2 编码，算法会误认为它们之间有数学联系（如 0<1<2）。独热编码将其转化为矩阵形式（0和1）。
    
- **模块**：`sklearn.preprocessing.OneHotEncoder`
    
- **核心代码**：
    
    Python
    
    ```
    from sklearn.preprocessing import OneHotEncoder
    
    # 假设 X 的第1,2列是分类变量
    X_cat = data.iloc[:, [1,2]]
    enc = OneHotEncoder(categories='auto').fit(X_cat)
    result = enc.transform(X_cat).toarray() # 必须 toarray() 才能看到矩阵
    
    # 获取特征名
    enc.get_feature_names_out()
    
    # 将哑变量合并回原数据集
    newdata = pd.concat([data, pd.DataFrame(result)], axis=1)
    ```

### 1.4 处理连续型特征：二值化与分段

#### 1.4.1 二值化 (Binarizer)

将连续型变量根据某个阈值（threshold）划分为0和1（例如：年龄大于30为1，小于等于30为0）。

- **模块**：`sklearn.preprocessing.Binarizer`
    
- **代码**：
    
    Python
    
    ```
    from sklearn.preprocessing import Binarizer
    # 设定阈值为30
    transformer = Binarizer(threshold=30).fit_transform(X) 
    ```
    

#### 1.4.2 分箱/离散化 (KBinsDiscretizer)

将连续型变量划分为多个类别（分箱）。

- **模块**：`sklearn.preprocessing.KBinsDiscretizer`
    
- **参数**：
    
    - `n_bins`：箱子数量
        
    - `encode`：编码方式（"onehot" 返回稀疏矩阵，"ordinal" 返回整数）
        
    - `strategy`：分箱策略（"uniform" 等宽，"quantile" 等频，"kmeans" 聚类）

## 🎯 2. 特征选择 (Feature Selection)

当数据预处理完成后，我们有可能会面临大量的特征。特征选择的目的是**寻找最优特征子集**，降维、提高模型速度并防止过拟合。`sklearn.feature_selection` 模块专门用于特征选择。

主要包含三种方法：**Filter过滤法、Wrapper包装法、Embedded嵌入法**。

### 2.1 Filter 过滤法

根据各种统计检验中的各项指标来选择特征。**独立于任何机器学习算法**。

#### 2.1.1 方差过滤 (VarianceThreshold)

- **原理**：优先消除方差为0（或极小）的特征。如果一个特征的方差为0，说明所有样本在该特征上的取值都一样，对区分样本没有任何帮助。
    
- **核心代码**：
    
    Python
    
    ```
    from sklearn.feature_selection import VarianceThreshold
    
    # 实例化，不填参数默认消除方差为0的特征
    selector = VarianceThreshold() 
    # 也可以设定阈值，如消除一半方差最小的特征：
    # threshold = np.median(X.var().values)
    # selector = VarianceThreshold(threshold=threshold)
    
    X_var0 = selector.fit_transform(X) 
    ```
    

#### 2.1.2 相关性过滤

评估特征与标签之间的相关性。如果不相关，则剔除。

- **卡方检验 (`chi2`)**：专门针对**离散型标签**（分类问题）。计算非负特征和标签之间的卡方统计量。
    
    Python
    
    ```
    from sklearn.feature_selection import SelectKBest
    from sklearn.feature_selection import chi2
    
    # 假设需要选出前300个与标签最相关的特征
    X_fschi = SelectKBest(chi2, k=300).fit_transform(X_var0, y)
    ```
    
- **F检验 (`f_classif`, `f_regression`)**：既可以做分类也可以做回归，捕捉特征与标签之间的**线性关系**。
    
- **互信息法 (`mutual_info_classif`, `mutual_info_regression`)**：捕捉特征与标签之间的**任何关系（包括线性和非线性）**。返回[0,1]之间的值，0表示完全不相关。
    

---

### 2.2 Embedded 嵌入法

让算法自己决定使用哪些特征。把特征选择和算法训练过程结合在一起。模型会计算出各个特征的权重或重要性（如树模型的 `feature_importances_`，线性模型的 `coef_`），然后根据设定阈值来选择。

- **模块**：`sklearn.feature_selection.SelectFromModel`
    
- **核心代码**（以随机森林为例）：
    
    Python
    
    ```
    from sklearn.feature_selection import SelectFromModel
    from sklearn.ensemble import RandomForestClassifier
    
    rfc = RandomForestClassifier(n_estimators=10, random_state=0)
    
    # 以随机森林为基模型，选择特征重要性高于指定阈值的特征
    # threshold 可填具体数值，也可填 "mean", "median"
    X_embedded = SelectFromModel(rfc, threshold=0.005).fit_transform(X, y)
    ```
    

---

### 2.3 Wrapper 包装法

与嵌入法类似，也是依赖于算法自身的选择，但它使用的是目标函数（通常是预测效果指标）来每次选择若干特征，或每次排除若干特征。代表算法是**递归特征消除法 (RFE, Recursive Feature Elimination)**。

- **原理**：使用一个基模型进行多轮训练，每轮训练后，消除若干权值系数最小的特征，再基于新的特征集进行下一轮训练，直到达到指定的特征数量。
    
- **模块**：`sklearn.feature_selection.RFE`
    
- **核心代码**：
    
    Python
    
    ```
    from sklearn.feature_selection import RFE
    from sklearn.ensemble import RandomForestClassifier
    
    rfc = RandomForestClassifier(n_estimators=10, random_state=0)
    # n_features_to_select: 最终要保留的特征数量
    # step: 每次迭代移除的特征数量
    selector = RFE(rfc, n_features_to_select=340, step=50)
    selector = selector.fit(X, y)
    
    X_wrapper = selector.transform(X)
    
    # 查看特征是否被选中的布尔矩阵
    # print(selector.support_)
    # 查看特征的重要程度排名 (1为最高)
    # print(selector.ranking_)
    ```

## 💡 3. 经验总结与选择指南

1. **预处理顺序**：一般先处理缺失值，然后做类别变量编码，最后做连续变量无量纲化（如果模型需要的话）。
    
2. **特征选择顺序**：
    
    - 必须先用方差过滤 (Filter)剔除方差为0的废料特征（计算极快）。
        
    - 如果计算资源充足，优先使用**互信息法**过滤不相关的特征。
        
    - **嵌入法 (Embedded)** 和 **包装法 (Wrapper)** 更加精确，能考虑特征之间的组合效应，但计算量极大（Wrapper最慢）。通常在过滤法将特征维度降到能承受的范围内后，再使用这两种方法进行精细选择。
