    JSP全称Java Server Pages，是一种动态网页开发技术。它使用JSP标签在HTML网页中插入Java代码。标签通常以<%开头以%>结束。
    JSP是一种Java servlet，主要用于实现Java web应用程序的用户界面部分。JSP通过网页表单获取用户输入数据、访问数据库及其他数据源，然后动态地创建网页。
    JSP是Servlet的扩展，在没有JSP之前，就已经出现了Servlet技术。Servlet是利用输出流动态生成HTML页面，包括每一个HTML标签和每个在HTML页面中出现的内容。

一、jsp注释：

​       1、显示注释语法(能够在客户端上查看的注释)：从HTML风格继承而来<!--     -->

​        2、隐式注释风格(不能够在客户端上查看的注释)：从Java风格继承而来 //        或者/*           */

​                                                                                              jsp自己的注释<%--                  --%>

二、Scriptlet(脚本小程序)   在jsp中一共有三种Scriptlet代码

​        1、<%   %>:Java脚本段，可以设定局部变量，编写语句

​        2、<%!      %>:声明，可以定义全局（成员）变量、方法、类

​        3、<%=     %>:表达式，数据一个变量或具体内容

三、include静态包含         

​           *<@%include file="   "%@>

​              特点：1、耦合性较高，不够灵活   2、将内容进行了直接的替换    3、不能出现同名变量

​                          4、静态包含只会生成一个源码文件，最终的内容全部在jspservice方法文件中

四、include动态包含

​           *<jsp:include page=""></jsp:include>

​           特点：1、动态包含相当于方法的调用    2、动态包含会生成多个源码文件

​                       3、可以定义同名变量                  4、效率高，耦合度低

​                注：当动态包含不需要传递参数时，include双标签之间就不要有任何内容，包括换行和空格
