## 实验课题
局域网ARP欺骗与HTTP流量嗅探原理分析
## 实验目的
1. 理解ARP协议的工作机制及其安全性缺陷
2. 掌握IP转发在中间人攻击中的作用
3. 观察明文协议在不安全网络环境下的风险，增强安全意识
---
## 一、实验原理剖析

### 1. ARP协议原理
**目的**：在局域网中将IP地址解析为对应的物理地址（MAC地址）

**关键设计**：
- 无连接、无状态：无需建立连接，直接广播请求，无身份验证机制

**工作流程**：
1. 主机A需与IP为192.168.88.135的主机B通信，但不知其MAC
2. A广播ARP请求报文："谁有192.168.88.135?请回复MAC地址!"
3. 局域网内所有设备收到广播，主机B发现IP匹配
4. B单播回复ARP响应："我是192.168.88.135，我的MAC是BB:BB:BB:BB:BB:BB"
5. A收到响应后，将映射关系存入ARP缓存表

**ARP协议的缺陷**：
- 无状态性：主机不验证收到的ARP响应是否是其之前请求过的

### 2. 中间人攻击原理
攻击者通过持续发送伪造的ARP响应包，欺骗靶机和网关：
- 欺骗靶机：告诉靶机"我是网关"，靶机的流量会发往攻击者
- 欺骗网关：告诉网关"我是靶机"，回给靶机的流量也会发往攻击者

### 3. IP转发的作用
- 默认情况：Linux内核会丢弃不是发往本地的数据包
- 开启ip_forward：攻击机会像路由器一样，将捕获到的靶机流量重新转发给真实网关
- 作用：维持靶机网络连接正常，避免攻击暴露

## 三、攻击流程

### 第一步：信息收集与环境检查
**确认网关IP和靶机是否存在**
1. 扫描局域网存活主机：`sudo fping -asg 192.168.88.136/24    # 扫描局域网存活主机`
2. 在靶机上查看当前ARP缓存表（观察实验前后变化）：`arp -a`
![[Pasted image 20251229202203.png]]
### 第二步：开启攻击机核心转发能力
**维持"中间人"身份而不破坏网络连接的关键**
`echo 1 | sudo tee /proc/sys/net/ipv4/ip_forward    #临时开启转发`
临时开启IP转发：
**IP转发作用对比**：
- **关闭（值为0）**：内核丢弃非本地数据包 → 靶机断网 → 攻击暴露
- **开启（值为1）**：内核转发数据包 → 靶机上网正常 → 攻击隐蔽

### 第三步：双向ARP欺骗
**完整截获上下行流量，需要同时欺骗靶机和网关**
1. 终端1：欺骗靶机（让靶机认为攻击机是网关）
   `sudo arpspoof -i eth0 -t 192.168.88.136 192.168.88.2`
2. 终端2：欺骗网关（让网关认为攻击机是靶机）
   `sudo arpspoof -i eth0 -t 192.168.88.2 192.168.88.136`
### 第四步：流量抓取与分析
1. 在Kali上启动Wireshark
2. 选择攻击网卡（如eth0）
3. 设置过滤器：`http.request.method == "POST"`
   - 原理：只关心靶机提交表单的行为
### 第五步：触发实验结果
1. 靶机访问：`http://192.168.88.135/login.php`
   `curl -X POST http://192.168.88.135/login.php   -d "user=admin"   -d "pass=P@ssword123"`
2. 靶机输入任意账号密码（如admin/P@ssword123）并提交
3. 分析：在Wireshark中查看HTTP POST请求，在HTML Form URL Encoded部分查看明文账号密码
   ![[Pasted image 20251229202425.png]]
4. 观察实验后的arp变化：发现getway（网关）的MAC地址变成了攻击机的MAC地址

## 关键知识点总结

### ARP协议安全缺陷
- 无身份验证机制，易受欺骗攻击
- 无状态设计，主机接受未经请求的ARP响应

### 中间人攻击核心要素
1. **ARP欺骗**：篡改目标设备的ARP缓存表
2. **IP转发**：维持网络连通性，避免被发现
3. **流量嗅探**：在数据传输路径上截获和分析数据

### 安全防护要点
- **加密传输**：HTTPS是防御流量嗅探的最有效手段
- **网络设备防护**：在企业网络层面实施ARP检测
- **终端加固**：在固定环境中使用静态ARP绑定

通过本实验，深刻理解了ARP协议的安全风险和中间人攻击的实现原理，增强了在实际网络环境中的安全意识。
