## 一、创建虚拟机
创建虚拟机->自定义->稍后按照操作系统->windows和版本->选择安装位置->设置密码->UEFI（后面会改）->默认处理机配置->2048MB->默认网络配置->IO控制默认->SCSI->默认选项->默认磁盘大小->后续均默认配置
==说明==：BIOS是传统的基本输入输出系统，较旧的引导方式，UEFI是统一可扩展固件接口，新一代引导方式。
==注意==：系统内存至少需要4G

## 四、安装Windows11操作系统
1. 打开Windows虚拟机，等待
   ![[Pasted image 20260124225651.png]]
2. 选择 **CDROM Drive** 从光驱启动安装
3. 中间根据实际情况选取
4. 出现密钥时选择**我没有密钥**
5. 选择Windows11专业版

| 功能              | 家庭版   | 专业版  |
| --------------- | ----- | ---- |
| Hyper-V 虚拟机     | ❌ 不支持 | ✅ 支持 |
| BitLocker 加密    | ❌ 不支持 | ✅ 支持 |
| 远程桌面主机          | ❌ 不支持 | ✅ 支持 |
| 组策略编辑器          | ❌ 不支持 | ✅ 支持 |
| Windows Sandbox | ❌ 不支持 | ✅ 支持 |
| 加入域/AD          | ❌ 不支持 | ✅ 支持 |
