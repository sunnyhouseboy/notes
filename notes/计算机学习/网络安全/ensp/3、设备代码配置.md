## 一、视图
### 1、认识配置视图
\<Huawei>用户视图 【查看运行状态】
\[Huawei] 系统视图 【配置设备的系统参数】
\[Huaiwei-GigabitEthernet0/0/1] 接口视图 【配置接口参数】
\[Huawei-ospf-1] 协议视图 【配置XX协议】
### 2、切换视图
\<Huawei>system-view    //进入系统视图
\[Huawei] interface GigabitEthernet 0/0/1  //进入接口视图
### 3、退回视图
1. quit 退回到上一个视图（逐级退回）
2. return或者快捷键Ctrl+Z 直接返回到用户视图
## 二、交换机基本配置命令
1. \<Huawei>  `display current-configuration`   //查看当前有效配置
2. \<Huawei>`display  version`   //查看系统版本信息 
3. \[Huawei]`sysname  SW1`  修改设备名为SW1
4. \[Huawei-GigabitEthernet0/0/1] `undo shutdown`  //开启接口
5. \[Huawei-GigabitEthernet0/0/1] `shutdown`   //关闭接口
6. \<Huawei> `save ``  //保存配置
7. \<Huawei>`reboot`  //重启设备
8. \[Huawei] `user-interface console 0` //进入终端进行配置
   \[Huawei-ui-console0] `authentication-mode password `  //启用密码认证
   \[Huawei-ui-console0] `set authentication password cipher 123456`  //设想要验证密码，在用户视图中直接敲入quit退出，再按回车键提示输入密码
9. \<Huawei>`language-mode Chinese` //切换语言模式为中文
10. \<Huawei> `undo terminal monitor` //取消终端提示
11. \<Huawei> `terminal monitor` //打开终端提示
12. \[Huawei]`user-interface console 0` //进终端进行配置
     \[Huawei-ui-console0] `idle-timeout 1` //设置闲置会话时间为1分钟
 13. \<Huawei>`reset saved-configuration`  //恢复出厂设置（需要重启）
## 三、远程控制交换机
![[Pasted image 20260205152456.png]]
### 1、给设备配置IP地址
SW1：
\[Huawei]`interface Vlanif 1` //进入虚接口Vlanif1
\[Huawei-Vlanif1]`ip address 192.168.10.100 24` //配置IP地址
PC：
\[Huawei]`interface Vlanif 1` //进入虚接口Vlanif1
\[Huawei-Vlanif1]`ip address 192.168.10.200 24` //配置IP地址
SW1：
\<Huawei>`display interface Vlanif 1` //查询IP地址信息
### 2、测试两台设备的网络连通性
SW1：\<Huawei>`ping 192.168.10.200`
PC：\<Huawei>`ping 192.168.10.100`
### 3、实施远程
SW1：
\[SW1]`user-interface vty 0 4` //进入远程配置视图，允许5个会话远程访问
\[SW1-ui-vty0-4]`authentication-mode password` //启用密码认证模式
\[SW1-ui-vty0-4]`set authentication password cipher 123456 `//设置密文
\[SW1-ui-vty0-4]`user privilege level 3` //设置权限级别，默认为权限0参观
PC：\<huawei>` telnet 192.168.10.100` //实现远程控制
## 四、路由器的基本配置
![[Pasted image 20260205155120.png]]
### 1、给PC1、PC2配置IP地址、子网掩码、网关地址
### 2、给路由器接口配置IP地址
配置PC1设备的网关地址：
\[Huawei]`interface GigabitEthernet 0/0/0` //进入接口视图
\[Huawei-GigabitEthernet0/0/0]`ip address 192.168.10.254 24` 
配置PC2设备的网关地址：
\[Huawei]`interface GigabitEthernet 0/0/1` //进入接口视图
\[Huawei-GigabitEthernet0/0/1]`ip address 192.168.20.254 24 `
确认IP地址是否配置成功:
\[Huawei-GigabitEthernet0/0/0]`display this`
\[Huawei-GigabitEthernet0/0/1]`display this`
\<Huawei>`display ip interface brief`//查看接口IP地址简短信息
### 3、验证结果：测试PC1和PC2的网络连通性
在PC1上进行操作：`ping 192.168.20.1`
在PC2上进行操作：`ping 192.168.10.1`
### 4、查看路由表
\<Huawei>`display ip routing-table`
## 操作技巧
1. 命令是不区分大小写的
2. 按？可以获取到帮助信息
   方法一：直接在视图上敲？（在该视图上能够敲的命令有哪些）
   方法二：问命令全拼 （例如：s？）
   方法三：问命令后面能跟哪些命令（例如：interface ？）
3. 命令可以简写
4. 按TAB键自动补全
