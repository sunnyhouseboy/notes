# 数据可视化

## 一、可视化基础

### 1. 可视化图表要素

- 标题
- 坐标轴
- 图形要素
- 图例
- 数据标签

### 2. 常用可视化工具

| 工具 | 说明 | 优点 | 局限 |
| :-- | :-- | :-- | :-- |
| Matplotlib | Python 最基础的可视化库 | 灵活强大、定制性强 | 代码较多、默认风格基础 |
| Seaborn | 基于 Matplotlib 的高级接口 | 风格美观、统计图方便 | 对简单图略显繁琐 |
| Pandas Plot | 快速图表，调用 `.plot()` | 快捷、适合 EDA | 图表样式较少 |

## 二、Matplotlib

### 1. 折线图（plot）

> 适用场景：观察数据随时间或顺序变化的趋势。

```python
import matplotlib.pyplot as plt
from matplotlib import rcParams

# 中文字体显示
# rcParams["font.family"] = "SimHei"
rcParams["font.sans-serif"] = ["SimHei"]

# 数据
months = ["一月", "二月", "三月", "四月"]
sales = [100, 150, 80, 130]

# 画布大小
plt.figure(figsize=(10, 5))

# 绘制图表
plt.plot(
    months,
    sales,
    label="产品A",
    color="orange",
    linewidth=2,
    marker="o",
    linestyle="--",
)

# 添加标题与坐标轴标签
plt.title("2025年销售趋势", color="red", fontsize=20)
plt.xlabel("月份", color="black", fontsize=10)
plt.ylabel("销售额（万元）", color="black", fontsize=10)

# 添加图例和网格线
plt.legend(loc="upper left")
plt.grid(True, alpha=0.5, color="black", linestyle="--")

# 设置刻度与 y 轴范围
plt.xticks(rotation=0, fontsize=10)
plt.yticks(rotation=0, fontsize=10)
plt.ylim(0, 160)

# 在每个数据点上显示数值
for x, y in zip(months, sales):
    plt.text(x, y, str(y), ha="center", va="bottom", fontsize=10)

# 自动优化排版并展示图表
plt.tight_layout()
plt.show()
```

### 2. 条形图（bar）

> 适用场景：比较不同类别之间的数量差异。

```python
import matplotlib.pyplot as plt
from matplotlib import rcParams

# 柱状图 bar（竖向），条形图 barh（横向）

# 中文字体显示
# rcParams["font.family"] = "SimHei"
rcParams["font.sans-serif"] = ["SimHei"]

# 数据
subjects = ["语文", "数学", "英语", "科学"]
scores = [100, 150, 80, 130]

# 画布大小
plt.figure(figsize=(10, 5))

# 绘制柱状图
plt.bar(subjects, scores, label=subjects, color="green", edgecolor="black")

# 添加标题与坐标轴标签
plt.title("各科成绩对比", color="red", fontsize=20)
plt.xlabel("科目", color="black", fontsize=10)
plt.ylabel("分数", color="black", fontsize=10)

# 添加图例和网格线
plt.legend(loc="upper left")
plt.grid(True, alpha=0.5, color="black", linestyle="--")

# 设置刻度与 y 轴范围
plt.xticks(rotation=0, fontsize=10)
plt.yticks(rotation=0, fontsize=10)
plt.ylim(0, 160)

# 在每个柱子上显示数值
for x, y in zip(subjects, scores):
    plt.text(x, y, str(y), ha="center", va="bottom", fontsize=10)

plt.tight_layout()
plt.show()
```

### 3. 饼图（pie）

> 适用场景：展示整体中各部分所占的比例。

```python
import matplotlib.pyplot as plt
from matplotlib import rcParams

# 中文字体显示
# rcParams["font.family"] = "SimHei"
rcParams["font.sans-serif"] = ["SimHei"]

# 数据
things = ["学习", "娱乐", "运动", "睡觉", "其他"]
times = [6, 4, 1, 8, 5]
colors = ["red", "green", "blue", "yellow", "magenta"]
explode = [0.1, 0, 0, 0, 0]

# 画布大小
plt.figure(figsize=(10, 5))

# 绘制图表
plt.pie(
    times,
    labels=things,
    autopct="%1.1f%%",      # 显示百分比
    shadow=True,            # 设置阴影
    startangle=90,          # 调整图像角度
    colors=colors,          # 调整颜色
    wedgeprops={"width": 0.6},  # 绘制圆环
    pctdistance=0.6,        # 调整百分比显示位置
    explode=explode,        # 设置突出块的位置
)

# 添加标题与中心文字
plt.title("一天时间分布", color="red", fontsize=20)
plt.text(0, 0, "总计100%", ha="center", va="bottom", fontsize=10)

plt.tight_layout()
plt.show()
```

### 6. 多个图表

> 适用场景：在一张画布中同时对比多种图表。

```python
import matplotlib.pyplot as plt
from matplotlib import rcParams

# 中文字体显示
# rcParams["font.family"] = "SimHei"
rcParams["font.sans-serif"] = ["SimHei"]

# 数据
months = ["1", "2", "3", "4"]
sales = [100, 150, 80, 130]

# 两行两列，共四个子图
fig, axes = plt.subplots(2, 2, figsize=(10, 6))

axes[0, 0].plot(months, sales)
axes[0, 0].set_title("折线图")

axes[0, 1].bar(months, sales)
axes[0, 1].set_title("柱状图")

axes[1, 0].scatter(months, sales)
axes[1, 0].set_title("散点图")

axes[1, 1].barh(months, sales)
axes[1, 1].set_title("条形图")

plt.tight_layout()
plt.show()
```

## 三、Seaborn

### 1. 常用统计图

> Seaborn 适合快速绘制统计图表，常用于探索性数据分析（EDA）。

```python
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt
from matplotlib import rcParams

# 中文字体显示
# rcParams["font.family"] = "SimHei"
rcParams["font.sans-serif"] = ["SimHei"]

# 读取并清洗数据
penguins = pd.read_csv(r"E:\数据练习\sales.csv")
penguins.dropna(inplace=True)

# penguins.info()
# print(penguins.head())

# 1. 直方图：kde=True 表示叠加核密度估计图
sns.histplot(data=penguins, x="product", kde=True)

# 2. 核密度估计图（KDE, Kernel Density Estimate Plot）
sns.kdeplot(data=penguins, x="unit_price")

# 3. 计数图：统计分类变量出现次数
# countplot 统计类别频数（离散数据），histplot 观察数值分布（连续数据）
sns.countplot(data=penguins, x="product")

# 4. 散点图：可通过 hue 参数设置不同组别进行对比
sns.scatterplot(data=penguins, x="date", y="unit_price", hue="product")

# 5. 蜂窝图：kind 设置为 hex
sns.jointplot(data=penguins, x="quantity", y="unit_price", kind="hex")

# 6. 成对关系图：
# 对角线通常显示每个变量的单变量分布；
# 其他位置展示变量之间的两两关系。
sns.pairplot(data=penguins, hue="product")

# 7. 二维核密度估计图
# 8. 条形图
# 9. 箱线图
# 10. 小提琴图

plt.show()
```
