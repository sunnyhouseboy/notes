    1、读取数据

    首先导入必要的库并读取网络流量数据集。

    import pandas as pd
import numpy as np

data = pd.read_csv("Tuesday-WorkingHours.pcap_ISCX.csv")
data.columns = data.columns.str.strip() # 去除标签前后的空格
````

# 2、清洗数据

检查数据集中的缺失值（Null）和无限大（Infinity）的值。

```python
data.isnull().sum()

    numeric_df = data.select_dtypes(include=[np.number])
np.isinf(numeric_df).sum()

    将无限大的数值替换为空值（NaN），以便统一处理。

    data.replace([np.inf, -np.inf], np.nan, inplace=True)

    缺失值处理策略：

    • 少量缺失，缺失率 < 1%（大概率是这种情况） → 直接删除

    • 中等缺失，缺失率 1%~10% → 中位数填充

    • 大量缺失，缺失率 > 30% → 考虑删除该列

    注意： 处理网络流量数据时，尽量不要优先使用均值填充。

    计算当前数据的缺失率：

    # 计算缺失率
print(f"{(data.isnull().sum().sum()/data.shape[0])*100}%")

    输出: 0.11840981007335577%

    因为缺失率极低，直接删除包含缺失值的行：

    data.dropna(inplace=True)

    3、标签映射

    将目标变量 Label 进行二值化处理，BENIGN（正常流量）映射为 0，其余异常流量映射为 1。

    data['Label'] = (data['Label'] != 'BENIGN').astype(int)

    4、特征选择

    特征选择的常用步骤：

    1. 手工删除明显不该参与训练的字段。

    2. 方差筛选，删掉几乎不变化的列。

    3. 相关系数筛选，看每个特征和 Label 的相关性，选相关性高的特征；或者使用 SelectKBest。

    4.1 手工删除明显无关的字段

    如果数据中包含以下身份或时间标识字段，可以直接删除。

    drop_cols = [
    'Flow ID', 'Source IP', 'Source Port',
    'Destination IP', 'Timestamp'
] # Destination Port 不能默认删除

data.drop(drop_cols, axis=1, inplace=True, errors='ignore') # axis=1 指定删除的对象是列

    4.2 方差筛选

    删除几乎不发生变化的列（方差为 0）。

    注意：VarianceThreshold 只能处理数值特征，若数据中有离散字符串类别，需先做编码（如 LabelEncoder 或 OneHotEncoder）。

    from sklearn.feature_selection import VarianceThreshold

selector = VarianceThreshold(threshold=0)
selector.fit(data)

# 查看并删除方差为0的列，~表示数组按位取反
data.columns[~selector.get_support()].tolist() 
data = data.loc[:, selector.get_support()]

    4.3 统计检验特征筛选

    ① 相关系数筛选（代码被注释）：

    着重评估单变量线性相关性。

    corr = data.corr(numeric_only=True)['Label'].abs()
temp = corr.sort_values(ascending=False).head(16)
features = temp.index.tolist() # remove是原地修改，会直接改列表本身，但不返回新列表
features.remove('Label')
y = data['Label']
data = data[features]

    ② 使用 SelectKBest：

    着重评估类别分离能力。

    原理： 利用单变量统计检验，独立评估每个特征与目标变量的相关性，完全不考虑特征间的组合效应。优点是计算快速、易解释，适合在高维数据中快速去除无关特征。

    from sklearn.feature_selection import SelectKBest, f_classif

# 指定评分函数为分类任务中的 ANOVA F 检验。k=15：保留得分最高的 15 个特征。
selector = SelectKBest(score_func=f_classif, k=15)
x = data.drop('Label', axis=1)
y = data['Label']

selector.fit(x, y)
features = x.columns[selector.get_support()].tolist()
data = data[features]

    4.4 划分数据集

    from sklearn.model_selection import train_test_split

# test_size=0.2 表示 20% 测试集和 80% 训练集，stratify=y 表示分层抽样
X_train, X_test, y_train, y_test = train_test_split(
    data,
    y,
    test_size=0.2,
    random_state=42,
    stratify=y
)

print(X_train.shape)
print(X_test.shape)
print(y_train.value_counts())
print(y_test.value_counts())

    5、特征缩放

    对特征数据进行标准化处理，使其均值为 0，方差为 1。

    from sklearn.preprocessing import StandardScaler

scaler = StandardScaler()

# 用训练集拟合并转换
X_train_scaled = scaler.fit_transform(X_train)

# 测试集只转换，不进行拟合，防止数据泄露
X_test_scaled = scaler.transform(X_test)

    6、模型训练

    使用逻辑回归（Logistic Regression）模型进行训练。考虑到样本类别极度不平衡，这里使用了 class_weight='balanced' 参数。

    from sklearn.linear_model import LogisticRegression

model = LogisticRegression(class_weight='balanced', max_iter=1000)
model.fit(X_train_scaled, y_train)

    7、模型评估

    from sklearn.metrics import accuracy_score

y_pred = model.predict(X_test_scaled)
acc = accuracy_score(y_test, y_pred)
print("准确率:", acc)

    输出: 准确率: 0.92579295178898

    10、模型拯救方案

    如果模型在少数类上表现不佳，可以尝试以下策略：

    # ① 阶级权重惩罚 (Class Weight)
model = LogisticRegression(class_weight='balanced')

    11、模型改进

    • 更换更强大的武器（算法升级）： 逻辑回归（Logistic Regression）本质上只能画出一条直的线（线性决策边界）。面对极其复杂的网络流量特征，直线太死板了。

      尝试使用树模型，比如随机森林（Random Forest）或 XGBoost。它们能画出高度非线性的、极其复杂的边界，能够精准地把交叠区的红色和蓝色剥离开来，从而在保持高 Recall 的同时，大幅提升 Precision！

    • 特征工程的魔法（降维打击）： 也许模型分不开红蓝，是因为我们给的特征不够好。你可以尝试去分析 CIC-IDS-2017 数据集的业务含义，丢掉那些干扰项，或者把几个特征组合成新的强力特征。

    • 高级数据平衡术（SMOTE 算法）： 与其给极少数的攻击流量加重罚款，不如我们在训练前，利用算法人工合成一些逼真的攻击流量数据（这叫过采样，比如著名的 SMOTE 算法），让模型在一个更平衡的平行宇宙里去学习。
